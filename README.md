## CAB420 Assessment 1A: Machine Learning Problem Solving

This repository contains the work for **Assessment 1A**, a problem-solving task focused on the application and analysis of various machine learning methods across different datasets. The project explores regression, classification, and deep learning techniques to solve real-world data challenges.

---

### Project Overview

The primary goal of this assessment is to apply machine learning principles to complex business, science, and health contexts while evaluating the performance and ethical considerations of each approach.

### Core Problems

#### 1. Regression Analysis (Socio-Economic Data)

* 
**Objective**: Predict violent crimes per capita using 1990 US census data.


* 
**Models**: Linear Regression, Ridge Regression, and LASSO.


* 
**Key Tasks**: Model validation using $\lambda$ selection and evaluation of predictive power versus model complexity.



#### 2. Multi-Class Classification (Land Use)

* 
**Objective**: Classify land types (e.g., 'Sugi' forest, 'Hinoki' forest) based on spectral reflectance properties from aerial sensors.


* 
**Models**: K-Nearest Neighbours, Random Forest, and an ensemble of Support Vector Machines (SVM).


* 
**Key Tasks**: Hyper-parameter optimization using grid search on validation data.



#### 3. Deep Learning (Street View House Numbers)

* 
**Objective**: Develop and adapt Deep Convolutional Neural Networks (DCNN) for image classification using the SVHN dataset.


* 
**Experiments**: Comparing models trained from scratch with and without data augmentation.


* 
**Key Tasks**: Analysis of training convergence, inference time, and the impact of limited training data.
